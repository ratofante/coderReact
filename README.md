# React Shop

Tienda ficticia como proyecto para el curso de [CoderHouse](https://www.coderhouse.com). Usando [pokeApi](https://pokeapi.co/).

## Para arrancar:

npm i
npm start

La página está montada en [Vercel](https://react-poke-shop.vercel.app/).
